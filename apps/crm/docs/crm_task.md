# CRM Task Context

A `crm_task` represents an actionable item, activity, or reminder within the CRM system.

## Lifecycle
Tasks help sales representatives manage their daily workflow. A task's lifecycle is managed by its `status` (Backlog, Todo, In Progress, Done, Canceled).

## Key Fields
- **Priority**: Determines urgency (Low, Medium, High).
- **Start Date / Due Date**: Defines the timeframe for the task.
- **Title / Description**: Details what needs to be accomplished.

## Usage
Tasks can be used standalone or linked implicitly to deals and leads through user workflows to ensure follow-ups and meetings are not missed.
